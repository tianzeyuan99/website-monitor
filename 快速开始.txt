========================================
中国海油网站监测工具 - 快速开始指南
========================================

【前提条件】
✅ Windows 电脑
✅ Python 3.8 或更高版本（如果没有，请先安装）
   下载地址: https://www.python.org/downloads/
   安装时务必勾选 "Add Python to PATH"

【打包步骤】

1. 将整个文件夹复制到 Windows 电脑

2. 双击运行 "一键打包.bat"

3. 等待打包完成（约 5-10 分钟）
   - 会自动安装依赖
   - 会自动安装 Playwright 浏览器
   - 会自动打包成 exe 文件

4. 打包完成后，在 dist 目录找到 "网站监测工具.exe"

【使用打包好的 exe】

1. 将 "网站监测工具.exe" 复制给用户（只需这一个文件）

2. 用户双击运行 exe

3. 浏览器会自动打开界面

4. 点击"开始监测"按钮

5. 等待监测完成，查看404链接结果

【常见问题】

Q: 提示"未找到Python"？
A: 需要先安装 Python 3.8+，安装时勾选 "Add Python to PATH"

Q: 打包失败？
A: 检查网络连接，确保能访问 pip 源

Q: exe 文件很大？
A: 正常，因为包含了所有依赖和浏览器引擎（约 100-200MB）

Q: 运行 exe 后没有自动打开浏览器？
A: 手动访问 http://127.0.0.1:5000
